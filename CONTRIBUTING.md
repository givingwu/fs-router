## 快速开始

```bash
# 克隆仓库
git clone https://github.com/givingwu/fs-router.git
cd fs-router

# 安装依赖
pnpm install

# 构建项目
pnpm build

# 运行测试
pnpm test

# 开发文档
pnpm docs:dev
```
